## Podcast와 Episode 관계 설정

- Podcast는 여러 개의 Episodes를 가질 수 있다.
- Episode는 무조건 하나의 Podcast 속한다.

<br>

## GraphQL API

- _모든 Podcasts 조회_
- _특정 Podcast 조회_
- _Podcast 생성_
- _Podcast 수정_
- _Podcast 삭제_

- _모든 Episodes 조회_
- _Episode 생성_
- _Episode 수정_
- _Episode 삭제_
